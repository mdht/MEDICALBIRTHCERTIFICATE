SCS Data Component Column

Notes

This column refers to the element of the SCS found in the expanded structure XML. The Doc: Logical Path column that it correlates with contains a hand-written path into the SCS structure which is parsed and navigated to in the SCS expanded structure XML to locate the matching data component.
