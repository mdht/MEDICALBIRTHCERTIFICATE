SCS DCM Column

Description

This is an experimental feature to demonstrate the ability to locate any DCMs that are re-used in the SCS in eXist, and to open the equivalent web page. Currently this rendering of the DCM does not show the original cardinalities, only the relationships to other DCMs referenced within the structure, and so it is not a useful tool for QA Testers who were presumed to be the primary users of this feature.
