DocBook Logical Path Column

Description

The text of any right-hand-side SCS Path that is asserted to implement an Information Requirement in the DocBook Requirments to SCS Mapping Table. The Requirement Number is not shown as a "Doc" column, as it is correlated with the Req: No column from the Requirements, and is shown only once under this heading in the table.

Text in italics is a comment, and not a true logical path navigation.
