UML Conformance Statement Column

Description

The Generated conformance text from the UML Element. Correlates the UML elements in the model with the text that MDHT generates for the constraints based on the UML class and the Severity and other tagged values attached to it by the CDA Profile. Dependent on the project having been built in MDHT, once built the ope. ning the uml.traces file automatically correlates the UML elements in the model with the text. This forms the basis for the text that will appear in the document at the UML: CDA IR Section number given in that column.
