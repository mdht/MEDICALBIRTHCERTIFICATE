UML Element Column

Description

The UML Model element that is correlated with an SCS Logical Data Component. May be created by an MDHT modeller or derived by the LMI from the Details File Mapping element for an SCS Data Component in combination with the SCS Tree context.

When added to the MDHT model manually, a new trace is stored by invoking a UML: Element cell context menu "Select UML Type Name for Tracing", and then choosing a corresponding SCS Data Component and clicking its context menu item "Create Trace between UML Type Name and SCS Data Component Name".
