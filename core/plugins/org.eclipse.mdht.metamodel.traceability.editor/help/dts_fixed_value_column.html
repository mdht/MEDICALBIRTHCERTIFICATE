Details File Fixed Value Column

Description

The fixed value specified for an SCS data component in an XPath expression in a <Mapping> element of a "Details" XML file. This information is used by the LMI to create an equivalent UML attribute default value.

This value should be used to create a UML default value for an attribute in CDA corresponding to the XPath which ends in "=fixed_value".

Correlates To

Not a calculated correlation in the Trace Matrix Tooling, but the LMI usually creates a corresponding UML: Value

Attribute Of

A substring of the XPath in a <Mapping> element of a DCM details file, ending in a "=fixed_value" constraint.

Cells Navigate To

The XML Details file which is the source of the value.
