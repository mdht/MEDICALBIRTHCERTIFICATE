UML Value Column

Description

This is the way in which MDHT represents fixed values from which constraints asserting a constant value for an attribute are represented. The Trace Matrix does not compute a correlation with Dts: fixed_value, but usually the LMI uses the value check part of XPath statements in the details file to set UML default values.

Key Status

N/A

Technology Space

MDHT CDA UML

Cardinality

0..1

Attribute Of

A meta-attribute of a UML Attribute type called "Default Value".

Cells Navigate To

If the value is from a vocabulary set, the link navigates to the UML Enumeration model of the vocabulary (subset) in MDHT. If it is not an enumerated value, then only its non-navigable literal representation is given.
