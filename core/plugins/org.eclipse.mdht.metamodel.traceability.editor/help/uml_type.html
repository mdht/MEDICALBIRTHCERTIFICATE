UML Type Column

Description

The CDA base type UML element that the MDHT Element is derived from. When the SCS is correlated to the CDA UML automatically by the LMI, this is the type selected by the LMI based on the SCS Component's Details File Mapping element as the base CDA Type for the UML Class or Association that is used to represent a Logical Model element. Alternatively, when the details file does not contain an appropriate Mapping element, a human modeller will choose a CDA Base type for the UML element from their modelling experience.
